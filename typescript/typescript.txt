## 1. Interface (TypeScript)

**Meaning:**
Interface defines the **shape of an object**.

**Example**

```ts
interface User {
  name: string;
  age: number;
}

let person: User = {
  name: "Pankaj",
  age: 22
};
```

**Use:**

* Object structure
* Class contract
* Extend interface

```ts
interface A {
  x: number;
}

interface B extends A {
  y: number;
}
```

---

## 2. Type (Type Alias)

**Meaning:**
Type creates a **new name for any type**.

**Example**

```ts
type User = {
  name: string;
  age: number;
};
```

**Union example**

```ts
type ID = string | number;
```

**Tuple example**

```ts
type Point = [number, number];
```

---

## 3. Difference

| Interface        | Type                       |
| ---------------- | -------------------------- |
| Used for objects | Used for any type          |
| Can extend       | Can use union/intersection |
| Used in OOP      | More flexible              |

Example:

```ts
interface Car {
  name: string;
}

type ID = string | number;
```

---

✅ **Simple rule**

* **Interface → object structure**
* **Type → any type (union, tuple, primitive)**
