

## BMI Calculator

A simple **BMI Calculator** using **HTML, CSS, and JavaScript**.

### Features

* Enter height (cm)
* Enter weight (kg)
* Calculate BMI
* Shows weight condition

### Tech Used

* HTML
* CSS
* JavaScript

### How to Use

1. Open `index.html`
2. Enter height and weight
3. Click **Compute BMI**
4. See BMI result
