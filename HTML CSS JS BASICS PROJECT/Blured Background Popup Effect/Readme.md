# Blurred Background Popup Effect

This is a small **HTML, CSS, JavaScript** project.
It shows a popup with **blurred background effect** when user clicks a button.

## Features

* Blurred background
* Popup open & close
* Simple UI
* Pure HTML, CSS, JS

## Files

* `index.html` – main HTML file
* `style.css` – design and styles
* `function.js` – popup logic

## How it works

* Click **“Click to join”** → popup opens
* Background becomes blurred
* Click **close icon (X)** → popup closes

## Technologies

* HTML
* CSS
* JavaScript

## How to run

1. Download or clone project
2. Open `index.html` in browser
3. Done ✅
